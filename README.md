# Nassau-Candy-Optimization
Data-driven supply chain optimization project using machine learning to predict lead time and improve product allocation efficiency.
# 🍬 Nassau Candy Supply Chain Optimization


## 📌 Problem Statement

The current supply chain system uses a static rule-based approach to assign products to divisions (factories). This results in:

* High delivery lead time
* Increased logistics cost
* Uneven workload distribution

---

## 🎯 Objective

To build a **data-driven model** that predicts lead time and optimizes product allocation across divisions and regions.

---

## 📊 Dataset Overview

The dataset includes:

* Product Name
* Division
* Region
* Ship Mode
* Order Date & Ship Date
* Sales & Profit

👉 A new feature **Lead Time** was created for analysis.

---

## ⚙️ Project Workflow

### 🔹 1. Data Preprocessing

* Cleaned column names
* Converted date columns
* Handled missing values

### 🔹 2. Exploratory Data Analysis

* Identified delays across regions
* Analyzed division performance

### 🔹 3. Feature Engineering

* Created lead time variable
* Encoded categorical data

### 🔹 4. Modeling

Models used:

* Linear Regression
* Random Forest
* Gradient Boosting

### 🔹 5. Evaluation Metrics

* RMSE
* MAE
* R² Score

---
